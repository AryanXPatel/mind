## How Pi Credits Work

::steps
### Every request costs π credits

Each API call deducts a flat π amount based on the model. No token math needed.

### Check live rates

Browse the [Model Marketplace](https://api.bluesminds.com/pricing){target="_blank"} or query the API:

```bash
curl https://api.bluesminds.com/v1/models \
  -H "Authorization: Bearer $BLUESMINDS_API_KEY"
```

### Credits never expire

Your 500 free π credits last forever. Paid plans add quota on top.
::
