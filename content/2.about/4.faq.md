## What is BluesMinds?

BluesMinds provides **one API key** for 897+ AI models across 19 providers with unified billing. It works with any OpenAI-compatible library — just change `base_url`.

## How do I get started?

::steps
### Sign up at the Console

Go to [api.bluesminds.com/console](https://api.bluesminds.com/console)

### Generate an API Key

Navigate to **Tokens → Create New Token**

### Set your environment variables

```bash
export BLUESMINDS_API_KEY="sk-your-key"
export BLUESMINDS_BASE="https://api.bluesminds.com/v1"
```
::

## What happens when a provider is down?

BluesMinds automatically routes to an alternate provider. If no provider is available for your model, you'll receive a `404` error.

## Common Errors

| Error | Cause | Fix |
|-------|-------|-----|
| `401` | Invalid or missing API key | Check `Authorization: Bearer sk-...` header; ensure no trailing slash on base URL |
| `404` | Model not available | Refresh model list with `GET /v1/models`; some models require paid plans |
| `429` | Rate limit exceeded | Apply exponential backoff; check `Retry-After` header |

## Does BluesMinds support vision?

Yes. Vision requires publicly accessible image URLs or base64-encoded data in the request.

## Does BluesMinds support tool/function calling?

Yes, for supported models like `gpt-4o` and `claude-sonnet-4-5`. See [Tool Calling](/features/tool-calling).
