**BluesMinds** is a unified AI gateway that provides a single OpenAI-compatible endpoint for hundreds of large language models across providers like OpenAI, Anthropic, Google, Mistral, DeepSeek, and more.

::callout{type="info" icon="lucide:lightbulb"}
**Base URL:** `https://api.bluesminds.com/v1`
::

## Why BluesMinds?

::card-group{cols=2}
  ::card
  ---
  title: Automatic Failover
  icon: lucide:refresh-cw
  ---
  When a provider is down, BluesMinds silently reroutes your request to an alternate channel — no code changes required.
  ::

  ::card
  ---
  title: Cost Optimization
  icon: lucide:trending-down
  ---
  Selects the most cost-effective provider for a given model when multiple channels are available.
  ::

  ::card
  ---
  title: Privacy Controls
  icon: lucide:shield
  ---
  Request/response bodies are not stored by default. Opt-in logging with full control over data retention.
  ::

  ::card
  ---
  title: Drop-in Compatibility
  icon: lucide:check-circle
  ---
  Works with existing OpenAI SDKs unchanged — just update `base_url`.
  ::
::

## Core Concepts

| Term | Description |
|------|-------------|
| **API Key** | Bearer token (`sk-...`) used for all LLM inference calls |
| **Session Token** | Short-lived token for management API operations |
| **Model ID** | Provider-prefixed string (e.g., `gpt-4o`, `claude-sonnet-4-5`) |

## Getting Started

::steps
### Create Your Account

Sign in at the [BluesMinds Console](https://api.bluesminds.com/console).

### Generate an API Key

Navigate to **Tokens → Create New Token**.

### Set Your Environment Variable

```bash
export BLUESMINDS_API_KEY="sk-your-key"
```

### Start Making Requests

```bash
curl https://api.bluesminds.com/v1/chat/completions \
  -H "Authorization: Bearer $BLUESMINDS_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```
::
