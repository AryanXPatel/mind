BluesMinds routes requests across multiple providers. Models are available based on your plan tier.

## Free Credits

::callout{type="info" icon="lucide:gift"}
Every new account receives **500 pi credits** on signup. Credits never expire.
::

## Discovering Models

::tabs{variant="card"}
  ::div{label="OpenAI-style" icon="lucide:terminal"}
  ```bash
  curl https://api.bluesminds.com/v1/models \
    -H "Authorization: Bearer $BLUESMINDS_API_KEY"
  ```
  ::

  ::div{label="Gemini-style" icon="lucide:terminal"}
  ```bash
  curl https://api.bluesminds.com/v1beta/models \
    -H "Authorization: Bearer $BLUESMINDS_API_KEY"
  ```
  Returns Gemini-style metadata with additional model capabilities.
  ::
::

::callout{type="warning" icon="lucide:alert-triangle"}
**Always read `/v1/models` before hard-coding IDs.** Model availability changes as providers update their offerings.
::

## Access Tiers

| Plan | Access Level |
|------|-------------|
| **Free** | Basic models, ~50% context window |
| **10-Day Pass** | Good quality models, full context |
| **Unlimited** | All models, full context |
| **Enterprise** | All models + priority routing |

## Usage

Pass your chosen model ID in the `model` field:

```python
response = client.chat.completions.create(
    model="claude-sonnet-4-5",  # Any model from /v1/models
    messages=[{"role": "user", "content": "Hello!"}],
)
```

::callout{type="warning" icon="lucide:alert-triangle"}
If you receive a `model_not_found` error, refresh your model list with `GET /v1/models` and update accordingly.
::
