[Codex CLI](https://github.com/openai/codex) is OpenAI's open-source agentic coding tool for the terminal. Route it through BluesMinds for access to cross-provider models.

## Prerequisites

:pm-install{name="-g @openai/codex"}

## Setup

::tabs{variant="card"}
  ::div{label="Environment Variables" icon="lucide:terminal"}
  ```bash
  export OPENAI_BASE_URL=https://api.bluesminds.com/v1
  export OPENAI_API_KEY=sk-your-bluesminds-key
  ```
  ::

  ::div{label="Inline" icon="lucide:play"}
  ```bash
  OPENAI_BASE_URL=https://api.bluesminds.com/v1 \
  OPENAI_API_KEY=sk-your-key \
  codex "your task"
  ```
  ::

  ::div{label="Config File" icon="lucide:settings"}
  Edit `~/.codex/config.json`:

  ```json
  {
    "model": "gpt-4o",
    "provider": "openai",
    "providers": {
      "openai": {
        "apiKey": "sk-your-bluesminds-key",
        "baseURL": "https://api.bluesminds.com/v1"
      }
    }
  }
  ```
  ::
::

## Recommended Models

| Model | Context | Best For |
|-------|---------|----------|
| `gpt-4o` | 128k | General coding tasks |
| `gpt-4.1` | 1M | Large codebase analysis |
| `gpt-4o-mini` | 128k | Fast, cost-efficient |
| `o3` | 200k | Complex reasoning |
| `o4-mini` | 200k | Balanced reasoning |

## Benefits

- Access cross-provider models (Claude, Gemini) through a single key
- Automatic failover when providers are rate-limiting
- Cost optimization via intelligent provider routing
- One bill for everything

## Troubleshooting

| Problem | Solution |
|---------|----------|
| `401` error | Confirm key starts with `sk-` |
| `404` error | Model IDs are **case-sensitive** — check `/v1/models` |
| Slow responses | Try `gpt-4o-mini` for faster completions |
| Cached base URL | Restart Codex after changing env vars |

**Connectivity test:**

```bash
curl https://api.bluesminds.com/v1/models \
  -H "Authorization: Bearer sk-your-key"
```
