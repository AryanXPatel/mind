[Claude Code](https://docs.anthropic.com/en/docs/claude-code) is Anthropic's AI coding assistant for the terminal. Redirect it through BluesMinds to get provider fallback, rate-limit distribution, unified billing, and privacy controls.

## Prerequisites

:pm-install{name="-g @anthropic-ai/claude-code"}

## Setup

Set two environment variables:

```bash
export ANTHROPIC_BASE_URL=https://api.bluesminds.com/
export ANTHROPIC_API_KEY=sk-your-bluesminds-key
```

::callout{type="info" icon="lucide:info"}
For persistent configuration, add these to your `~/.bashrc`, `~/.zshrc`, or shell profile.
::

## Model Selection

```bash
claude --model claude-sonnet-4-5
```

| Model | Use Case | Context |
|-------|----------|---------|
| `claude-opus-4-5` | Complex tasks, deep reasoning | 200k |
| `claude-sonnet-4-5` | Balanced quality/speed **(recommended)** | 200k |
| `claude-haiku-3-5` | Fastest, most cost-efficient | 200k |

## Troubleshooting

| Problem | Solution |
|---------|----------|
| `401 Unauthorized` | Verify key starts with `sk-`; check with `echo $ANTHROPIC_API_KEY` |
| `Connection refused` | Check URL formatting — ensure no trailing path components |
| Wrong endpoint | Restart Claude Code after changing env vars (it caches the base URL) |
| Older versions | May need `ANTHROPIC_API_URL` in addition to `ANTHROPIC_BASE_URL` |

**Connectivity check:**

```bash
curl https://api.bluesminds.com//models \
  -H "Authorization: Bearer sk-your-key"
```

::callout{type="info" icon="lucide:info"}
BluesMinds automatically maps the Anthropic message format to and from the appropriate upstream provider.
::
