[OpenClaw](https://openclaw.ai) is an AI agent platform with a built-in gateway. Since it supports OpenAI-compatible providers, you can route it through BluesMinds to access 100+ models with a single key.

## Configuration

Edit `~/.openclaw/openclaw.json`:

```json5
{
  // BluesMinds as the provider
  "mcpServers": {},
  "providers": {
    "bluesminds": {
      "baseUrl": "https://api.bluesminds.com/v1",
      "api": "openai-completions",
      "apiKeyEnv": "BLUESMINDS_API_KEY"
    }
  },
  "models": [
    {
      "id": "gpt-4o",
      "provider": "bluesminds",
      "contextWindow": 128000,
      "maxOutputTokens": 16384
    },
    {
      "id": "claude-sonnet-4-5",
      "provider": "bluesminds",
      "contextWindow": 200000,
      "maxOutputTokens": 8192
    },
    {
      "id": "gemini-2.0-flash",
      "provider": "bluesminds",
      "contextWindow": 1048576,
      "maxOutputTokens": 8192
    }
  ]
}
```

Set your API key:

```bash
export BLUESMINDS_API_KEY=sk-your-key
```

## Troubleshooting

| Problem | Solution |
|---------|----------|
| `401` errors | Verify key starts with `sk-` |
| Wrong models showing | Switch `mode` to `"replace"` instead of `"merge"` |
| Gateway issues | Run `openclaw onboard --install-daemon` |
| Config not loading | Use JSON5 format; validate with a linter |
