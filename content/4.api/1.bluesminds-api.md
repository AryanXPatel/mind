## Base URL

```
https://api.bluesminds.com/v1
```

## Authentication

| Surface | Auth Method | Example |
|---------|------------|---------|
| **LLM endpoints** | Bearer token | `Authorization: Bearer sk-...` |
| **Management** | Session token | Obtained via login |

## Core Endpoints

### Chat Completions

```bash
POST /v1/chat/completions
```

```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.bluesminds.com/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}],
)
```

### List Models

```bash
GET /v1/models
```

Returns all available models for your plan tier.

### Anthropic-style Messages

```bash
POST /v1/messages
```

For apps built against the Anthropic SDK — accepts native Anthropic message format.

### Image Generation

```bash
POST /v1/images/generations
```

## Error Handling

### Response Format

**Success:**

```json
{
  "id": "chatcmpl-abc123",
  "object": "chat.completion",
  "model": "gpt-4o",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "Hello! How can I help?"
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 10,
    "completion_tokens": 15,
    "total_tokens": 25
  }
}
```

**Error:**

```json
{
  "error": {
    "message": "Model not found",
    "type": "invalid_request_error",
    "code": "model_not_found"
  }
}
```

### Error Codes

| Status | Code | Action |
|--------|------|--------|
| `401` | Unauthorized | Check API key format and header |
| `403` | Forbidden | Key lacks permission for this resource |
| `404` | Not found | Model unavailable — refresh `/v1/models` |
| `429` | Rate limited | Backoff with jitter; check `Retry-After` |
| `500` | Server error | Retry with exponential backoff |

::callout{type="warning" icon="lucide:alert-triangle"}
**Retry logic:** Retry on `429` and `500` with exponential backoff. Never retry `401`, `403`, or `404` without fixing inputs.
::

## SDK Compatibility

BluesMinds is compatible with any OpenAI SDK by setting `base_url`:

::tabs{variant="card"}
  ::div{label="Python" icon="lucide:code"}
  ```python
  from openai import OpenAI
  client = OpenAI(
      api_key="sk-...",
      base_url="https://api.bluesminds.com/v1"
  )
  ```
  ::

  ::div{label="Node.js" icon="lucide:file-code"}
  ```javascript
  import OpenAI from "openai";
  const client = new OpenAI({
    apiKey: "sk-...",
    baseURL: "https://api.bluesminds.com/v1",
  });
  ```
  ::

  ::div{label="cURL" icon="lucide:terminal"}
  ```bash
  curl https://api.bluesminds.com/v1/chat/completions \
    -H "Authorization: Bearer sk-..." \
    -H "Content-Type: application/json" \
    -d '{"model":"gpt-4o","messages":[{"role":"user","content":"Hi"}]}'
  ```
  ::
